# Changelog

## 1.0.0 (2026-03-07)

Initial release.

### Skills

- **write-a-prd** — Create a PRD through deep interviewing and codebase exploration
- **prd-to-plan** — Break a PRD into phased implementation plan (tracer bullets)
- **prd-to-issues** — Fast path: PRD straight to GitHub issues
- **plan-to-issues** — Convert plan phases into GitHub issues
- **ralph** — Autonomous coding loop (setup, once, afk, team-research, team-review)
- **review-to-issues** — Parse PR review findings into GitHub issues for next cycle
- **grill-me** — Stress-test any plan or design through relentless interviewing
- **design-an-interface** — Generate multiple radically different interface designs
- **tdd** — Red-green-refactor test-driven development loop
- **request-refactor-plan** — Plan a refactor as tiny commits with testing strategy

### Agents

- **issue-researcher** — Read-only pre-implementation investigation (for team-research)
- **pr-reviewer** — Multi-lens PR review: correctness, security, architecture (for team-review)

### Hooks

- **TeammateIdle** — Validates structured output before teammate goes idle
- **TaskCompleted** — Validates structured findings before task marked complete
